# Alias

## Install
  ```shell
  cd ~
  git clone https://github.com/Justin-lu/alias.git .alias
  ```

## Usage
  Add the code below to your `.zshrc` or `.bashrc`

  ```shell
  # system shortcuts
  source ~/.alias/system.alias
  alias open-system-alias='vim ~/.alias/system.alias'

  #file shortcuts
  source ~/.alias/file.alias
  alias open-file-alias='vim ~/.alias/file.alias'

  # rails
  source ~/.alias/rails.alias
  alias open-rails-alias='vim ~/.alias/rails.alias'

  # ssh
  source ~/.alias/ssh.alias
  alias open-ssh-alias='vim ~/.alias/ssh.alias'

  # Heroku alias for ezitask
  source ~/.alias/heroku.alias
  alias open-heroku-alias='vim ~/.alias/heroku.alias'

  # Git alias
  source ~/.alias/git.alias
  alias open-git-alias='vim ~/.alias/git.alias'

  #alias for ioinc
  source ~/.alias/ionic.alias
  alias open-ionic-alias='vim ~/.alias/ionic.alias'

  # es6
  source ~/.alias/es6.alias
  alias open-es6-alias='vim ~/.alias/es6.alias'
  ```
